# Stock Submission

To submit your gifted stock, create a new file in this directory.

## Instructions
1.  Create a new file named `[YourName].json` (e.g., `Maxim.json`).
2.  Paste the following content and fill in the details:

```json
{
  "participant": "Maxim",
  "stockSymbol": "GME",
  "entryPrice": 25.00
}
```

3.  Commit and push the file.
4.  The system will automatically process it, add it to the portfolio, and delete your file.
